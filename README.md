# earn-auto-reviewer

AI-powered GitHub review system for Superteam Earn bounty submissions - automated scoring and feedback for PRs and repositories

This project provides an intelligent automated review system for GitHub submissions on the Superteam Earn platform. It analyzes pull requests and repositories to generate scores and actionable feedback.
